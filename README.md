# Smart-Meeting-Room-Assistant

Smart Meeting Room Assistant – PSOC™ Edge E84
🚀 Overview

This project was developed during the Infineon Hackathon using the PSOC™ Edge E84 evaluation kit.
The goal was to design a smart meeting room system that responds to touch, presence, and voice concepts to simplify room booking and automation.

🎯 Problem Statement

Design a smart meeting room assistant that:

Simplifies booking

Displays room availability

Auto-releases unused rooms

Provides real-time feedback

Enhances workplace automation

🛠 Tech Stack

PSOC™ Edge E84 Microcontroller

Embedded C

LVGL Graphics Library

Touch Interface

On-board Sensors

DEEPCRAFT™ Edge AI Concepts
